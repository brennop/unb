# Percursos de Euler

## Circuito de Euler:

- inicia e termina no **mesmo** vértice
- inclui cada vértice pelo menos uma vez
- inclui cada aresta **somente** uma vez


## Trilha de Euler 

- inicia e termina em vértices **diferentes**
- inclui cada vértice pelo menos uma vez
- inclui cada aresta **somente** uma vez
